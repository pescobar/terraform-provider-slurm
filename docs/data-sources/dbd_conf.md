# slurm_dbd_conf (Data Source)

Reads the active slurmdbd configuration — the live equivalent of
`sacctmgr show config`. Useful for preflight assertions on accounting
settings such as `TrackWCKey` or the purge windows.

~> Requires Slurm **26.05 or later** with `api_version = "v0.0.45"` (or
later). With an older `api_version` the plan fails with an error naming the
required versions; with a new enough `api_version` but an older cluster, the
error points at the server version instead.

## Example Usage

```terraform
# Read the active slurmdbd configuration — the live equivalent of
# `sacctmgr show config`. Requires Slurm 26.05+ (API version v0.0.45+).
data "slurm_dbd_conf" "accounting" {}

output "wckey_tracking_enabled" {
  value = data.slurm_dbd_conf.accounting.conf["TrackWCKey"]
}
```
